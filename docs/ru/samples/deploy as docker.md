## Инструкция по установке и запуску приложения Faceplate в docker контейнере

Требование: На вашем компьютере должен быть установлен и запущен Docker Engine.

Шаг 1: Создайте папку для проекта

Сначала создайте папку на компьютере, где будет «жить» приложение. В ней будут храниться базы данных и логи, чтобы они не потерялись.

Откройте Терминал и выполните команды:

mkdir my-faceplate-app
cd my-faceplate-app

Шаг 2: Создайте папки для данных

Внутри этой папки создайте две подпапки. Это позволит вам видеть логи и
файлы базы данных прямо на вашем компьютере.

mkdir logs
mkdir db

Шаг 3: Запустите приложение

Скопируйте весь блок кода ниже, вставьте его в терминал и нажмите Enter. Эта
команда скачает приложение и запустит его.

docker run -d \
-p 9000:9000 \
-p 7080:7080 \
-v $(pwd)/logs:/opt/faceplate/logs \
-v $(pwd)/db:/opt/faceplate/DB \
--name faceplate-instance \
fpcloud/faceplate:latest

Шаг 4: Проверьте, что всё работает

1. Убедитесь, что контейнер запущен, командой:
docker ps

2. Откройте браузер и перейдите по адресу: http://localhost:9000

Краткое пояснение команды (по желанию)

-d: Запускает приложение в фоновом режиме (чтобы терминал не
завис).

-p 9000:9000: Соединяет порт 9000 на вашем компьютере с
приложением.

-v ...: «Привязывает» папки, которые вы создали (logs и db), к
контейнеру. Это гарантирует, что ваши данные сохранятся даже при
перезапуске приложения.

fpcloud/faceplate:latest: Указывает Docker, какую именно версию
нужно скачать.
